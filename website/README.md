# Ace#1 Website - Complete FTP Upload Package

This directory contains a fully merged and FTP-ready version of the Ace#1 website, combining all resources from the 5 original zip files.

## Website Structure

### Main Pages
- `index.html` - Home page
- `our-products.html` - Products page  
- `ace-gallery.html` - Gallery page
- `about-us.html` - About page
- `contact-us.html` - Contact page

### Supporting Files
- `manifest.webmanifest` - Web app manifest
- `sw.js` - Service worker for PWA functionality

### Resource Directories
- `img1.wsimg.com/` - Images and media assets
- `fonts.googleapis.com/` - Google Fonts
- `fonts.gstatic.com/` - Font files
- `drive.google.com/` - Google Drive content
- `www.youtube.com/` - YouTube embeds
- `maps.googleapis.com/` - Google Maps integration
- `_DataURI/` - Base64 encoded data
- Various CDN directories for external resources

## Deployment Instructions

1. Upload the entire `website` directory contents to your web server's public folder
2. Ensure the web server can serve all file types (.js, .css, .webp, .svg, etc.)
3. Set up proper MIME types if needed
4. The website should be fully functional with all assets and functionality preserved
